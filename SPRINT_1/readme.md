## :computer: Requisitos para instalação do projeto


**Instalação:**

- Create your venv

```
$ python -m venv venvname
```

- Activate your venv
```
$ ./venvname/Scripts/activate
```

- Clone the repository
```
$ git clone https://github.com/Time-1-ADS/ProjetoGSW
```
